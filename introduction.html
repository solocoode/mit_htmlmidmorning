Content

Information stored inside the tag brackets is called an attribute.

Information inside the tags is called an element.

A doctype is a declaration of the version of Html being used. The above doctype declares that this is Html 5.

(Lang=en)is an attribute whose function is just to tell us what language we are coding the website in.

The (head) contains important information about the website . This is where the metadata is kept amd read. It is also where the style is coded in the case that there is no css page.

(Title) is used to store how the website is to be called.

(Body) contains the information in the website itself.

(hr) tags are used to separate the website into segments or sectors.

Meta tags are used to adjust the website as follows.

(meta http-equiv="X-UA-Compatible" content="ie=edge")This makes our website compatible with all available browsers.

(meta name="viewport" content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0") This enables the website to automatically adjust accordingly to the type and size of the device being used to access it.
